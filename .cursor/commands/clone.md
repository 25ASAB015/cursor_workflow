# /clone

Propósito: Clonar un repositorio usando `try` en `~/src/tries` y abrirlo en una nueva ventana de Cursor.

Parámetros:
- repo (HTTPS/SSH; acepta prefijo `@`)
- name (nombre de carpeta destino)

Ejecución:
```bash
./.cursor/bin/clone.sh "{repo}" "{name}"
```

Ayuda completa y ejemplos:
```bash
./.cursor/bin/clone.sh --help
```
